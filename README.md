## Quick Start

Add to your MCP client config:

```json
{
  "mcpServers": {
    "dns": {
      "url": "https://gateway.pipeworx.io/dns/mcp"
    }
  }
}
```

Or use the CLI:

```bash
npx pipeworx use dns
```
